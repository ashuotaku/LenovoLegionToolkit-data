# 📗 Readme:
These are the settings/data for [LenovoLegionToolkit](https://github.com/BartoszCichecki/LenovoLegionToolkit) for my gaming laptop **Lenovo LOQ 15IRX9**.
## If you have a similar laptop, then follow these simple steps:
1. Download this repo as zip and extract it and copy the files inside it.
2. Then press `Win + R` and type this `%USERPROFILE%\AppData\Local\LenovoLegionToolkit` and press enter.
3. Paste the copied files in the opened folder.
4. Restart your LenovoLegionToolkit/Laptop.
